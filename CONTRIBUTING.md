# Contributing

Thanks for helping improve Agentic Cockpit.

## Development
- Node >= 20
- Run tests: `npm test` or `pnpm test` or `node --test`

## Guardrails
- Do not commit secrets (API keys, tokens, credentials).
- Keep logs and fixtures secret-free.
- Prefer small, reviewable changes with tests.

## Reporting issues
Include:
- OS / WSL info
- `codex --version`
- minimal repro steps
